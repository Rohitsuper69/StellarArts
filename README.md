# StellarArts

A website for selling and buying digital arts online. Browse through a range of products and you can also sell your digital goods. 

## Tech Stack
 - HTML
 - CSS
 - JavaScript
 - Python
## Framework
 - Django
 - Tailwind
 - Flowbite
